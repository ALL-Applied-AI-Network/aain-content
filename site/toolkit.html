01 — Your chapter website

Hub Website Template

Go from fork to live site in under 10 minutes. One config file. Zero backend.

1

Fork the template

Click "Use this template" on GitHub. You get a full website repo with dark theme, event pages, leaderboards, and sponsor showcase.

2

Edit hub.config.json

Set your chapter name, university, colors, officers, and social links. Everything on the site updates automatically from this one file.

3

Push & deploy

Push to main and GitHub Actions auto-deploys to GitHub Pages. Your chapter has a live website with zero DevOps work.

Everything your chapter needs, pre-built and ready to deploy

The hub template gives every chapter a polished web presence out of the box — complete with curriculum integration, member tracking, and sponsor showcases.

  • Static site — HTML/CSS/JS, deployable to GitHub Pages, Vercel, or any host
  • Dashboard API integration — connect an API key and your events, members, and leaderboard sync live
  • Learning tree embed — the full curriculum with progress tracking, right on your site
  • Embeddable leaderboard — gamified progress and contribution tracking via iframe or API
  • CI/CD included — GitHub Actions workflow auto-builds and deploys on every push to main
