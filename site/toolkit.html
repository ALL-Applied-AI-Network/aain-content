03 — Infrastructure

Hub Website Template

Spin up a chapter website in minutes with the official hub template

Everything your chapter needs, pre-built and ready to deploy

The hub template gives every chapter a polished web presence out of the box — complete with curriculum integration, member tracking, and sponsor showcases. Fork it, customize it, deploy it.

  • Deployable site — static HTML/CSS/JS, works on GitHub Pages or any host
  • Learning tree integration — embed the full curriculum with progress tracking
  • Member leaderboards — gamified progress and contribution tracking
  • Sponsor showcase — highlight industry partners and attract new sponsors

npm run dev
